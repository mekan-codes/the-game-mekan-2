04_moving_dock_heavy.png - Moving dock / mobile delivery platform
05_heavy_hazard_container.png - Heavy cargo container
06_precision_fragile_cargo.png - Precision fragile cargo crate
07_hydraulic_crusher.png - Hydraulic crusher / press hazard
08_extendable_bridge_platform.png - Extendable bridge / telescoping platform
09_laser_security_gate.png - Laser security gate
10_rotating_sweeper_arm.png - Rotating sweeper arm hazard

Suggested use:
- Use the cargo assets as alternate mission cargo.
- Use the mechanical hazards for advanced timing / precision levels.
- Prefer scaling these in-game rather than stretching them.
